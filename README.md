This is a practice project for playing around with the `react-native`.

# What does this app do
Just some basic features:
- navigation
- sign-in form
- a dashboard with user data fetching from [randomuser.me](https://randomuser.me/)

# Commands

- `start` - start expo bundling.
- `android` - start android simulator.
- `ios` - start ios simulator.
- `web` - start web version.
- `eject` - to eject as an raw configs rather than keeping this boilerplate.
- `test` - run test in watch mode.
- `test:ci` - run test once and generate a coverage report.

# Tools

- `expo` - one of the boilerplates & platform for `react-native`.
- `native-base` - an out-of-the-box UI library for `react-native`.
- `formik` - a form library for React.
- `redux-starter-kit` - a UI state management tool for lazy person coming from `redux`.
- `redux-saga` - a flow control library for enhancing the redux.
- `husky` - run automation setup before commiting your code.
- `lintstage` - run linting and other tools for your git staged codes.
